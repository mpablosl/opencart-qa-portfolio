##  Objetivos de Teste

*OBJ-001 - Validar que somente clientes autenticados, com pelo menos um produto no carrinho, possam acessar o processo de checkout

*OBJ-002 - Validar que o usuário autenticado não consiga avançar no processo de checkout sem preencher os campos obrigatórios de entrega

*OBJ-003 - Validar que o usuário autenticado não consiga avançar no processo de checkout sem antes selecionar o método de envio

*OBJ-004 - Validar que o usuário autenticado não consiga avançar no processo de checkout sem antes selecionar o médodo de pagamento

*OBJ-005 - Validar o cálculo do valor total do pedido, incluindo frete, impostos e descontos

*OBJ-006 -  Validar se o sistema exibte a confirmação do pedido antes de realizar o Checkout

*OBJ-007 - Validar a confirmação do pedido (order) após o checkout
